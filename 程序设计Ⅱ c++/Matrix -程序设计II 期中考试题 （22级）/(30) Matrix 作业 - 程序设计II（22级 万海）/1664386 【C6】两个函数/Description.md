# 【C6】两个函数

## 问题描述

现有两个函数$f$和$g$，当$x<10$时，$f(x)=g(x)=1$；当$x\geq 10$时，其表达式如下

$$
f(x)=f(x-1)g(x-2)+f(x-2)g(x-1) \\
g(x)=f(\lfloor x/2 \rfloor)g(x-1) \\
$$

请实现计算函数$f$的程序，即输入$x$求$f(x)$。由于结果可能过大，因此输出取模10007后的结果

## 输入格式

输入一个整数$x$

## 输出格式

输出一个整数$f(x)$模10007后的结果

## 数据范围

输入整数$x\in[1,10000]$

## 样例输入

```
20
```

## 样例输出

```
233
```
